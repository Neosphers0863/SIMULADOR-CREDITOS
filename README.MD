<h1 align="center">Documentación Proyecto</h1>

###

<p align="center">Este proyectó muestra la construcción de una aplicación web estática manipulada con JavaScript y simulando el comportamiento de datos en un arreglo o archivos Json.<br>El proyecto se desarrolla bajo el patrón de diseño en MVC (Model-View-Controller).</p>

###

<h2 align="center">Instalación y/o Ejecución</h2>

###

<p align="center">- Clonar el repositorio.<br>- Abrir el proyecto con el editor preferido (Este proyecto se desarrollo con Visual Studio Code).<br>- Ejecutar el archivo index.html.</p>
